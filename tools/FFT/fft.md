# 用Python代码实现声音的时域转频域的变换

## 流程

- 弹出窗口, 询问声音文件的存储路径.
- 导入声音文件, 并进行快速傅里叶变换, 将声音的时域文件转换为频域文件.并进行A计权处理. 显示从0Hz~20000Hz的数据.每10Hz取1组数据
- 询问数据的保存路径, 并按照录入的文件名+年月日格式保存为新的文件, 格式为csv格式.
- 用matploylib导出频谱曲线.横坐标为0~20000Hz, 纵坐标为声音的幅值dB(A). 纵坐标的最大值为频谱中所有幅值中的最大值+5dB(A).
- 曲线为光滑蓝色细实线. 显示gird, 显示legend, 显示横坐标Title"Frequency(Hz)", 显示纵坐标(mm/s),其中峰值数据显示出频率和噪音值.
- 将生程度的曲线存入csv文件数据的右侧, 并另存为xlsx格式的同名文件.
