# My Skill

Auto commit change to Github. Pull the remote branch, merge the new change, and push up to remote branch

## When to Use

- When the project has change and 30mins no commit.
- This skill is helpful for record the change and avoid lost information.

## Instructions

- run git add .
- run git commit -m "<date time, like 2026-1-11 15:30> with <append version number like -001, -002 etc>"
- run git push - u origin office
